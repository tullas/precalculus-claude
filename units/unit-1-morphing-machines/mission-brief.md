# Sector 01 — Morphing Machines

**Mission:** The camera rig is broken. Match the target curve using four sliders.

**Key idea:** `g(x) = a·f(b(x − h)) + k`

- `a` → vertical zoom (flips if negative)
- `b` → horizontal zoom (flips if negative)
- `h` → horizontal shift (opposite sign of what you'd guess)
- `k` → vertical shift (matches what you'd guess)

**Your task:** Match the amber curve to the teal ghost curve using the four sliders.

**XP**

| Action | XP |
|---|---|
| Move each slider once | 10 |
| Match a target curve | 25 |
| Match 3 in one session | +15 bonus |

**Badge — Camera Calibrated:** match 3 targets in a row.
