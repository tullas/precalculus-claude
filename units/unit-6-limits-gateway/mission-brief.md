# Sector 06 — Limits: Gateway to Calculus

**Mission:** The sensor throws ERROR at one input. Report what it would say if it weren't broken there.

**Key idea:**

- A limit asks: as the input approaches `a`, what does the output approach? The function doesn't need to be defined at `a`.
- `0/0` means "wrong tool," not "no limit." Factor and cancel to find the real trend.

**Your task:** Zoom into the discontinuity from both sides. Estimate the limit, then verify by factoring.
